An extra margin of components added to every new work order to cover production spillage — machine set-up, dropped parts and AOI rejects. Each component's margin is the larger of the percentage and the minimum, rounded up to whole parts. So a small batch still gets at least the minimum (e.g. 15 spare of each part), while a large batch is driven by the percentage. The finished assembly quantity is unchanged. Leave both blank/0 for no margin. The values in effect are recorded on each work order at creation, so changing them here doesn't alter existing batches.
