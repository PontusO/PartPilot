PartPilot connects to the WooCommerce store using a REST API key. Stock sync is two-way — it reads product stock and writes reconciled quantities back — so the key needs Read/Write permission. Create one in WooCommerce under WooCommerce → Settings → Advanced → REST API.

Leave blank unless the shop runs a multi-currency plugin that returns prices in the wrong currency. Set your base currency code (e.g. SEK) to send ?currency=… on price reads. PartPilot already prefers the stored base-currency price fields, so this is only a fallback for plugins that need the hint.
